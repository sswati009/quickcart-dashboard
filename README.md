🛒 QuickCart Dashboard
📌 Problem Statement

Build a product listing dashboard for an e-commerce platform that:

Fetches products from an external API
Displays them in a clean UI
Allows filtering, searching, sorting, and basic interaction
Handles errors and supports responsive design
🚀 Features Implemented
Fetch products from FakeStore API
Display products in grid layout
Filter by category
Real-time search by product name
Sort by price (Low → High / High → Low)
View product details
Add to cart (UI counter)
Error handling with fallback message
Responsive design using Bootstrap

Challenges Faced
Managing original vs filtered product data
Implementing search without breaking filters
Handling async API calls properly
Keeping UI responsive across devices
