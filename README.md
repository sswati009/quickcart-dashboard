# 🛒 QuickCart Dashboard

A responsive product listing dashboard built using **HTML, JavaScript, and Bootstrap**, designed to simulate a basic e-commerce admin panel.

This project fetches real-time product data from an external API and allows users to interact with it through filtering, searching, sorting, and cart simulation features.

## 🚀 Features

* 🔹 Fetch products from FakeStore API
* 🔹 Display products in a clean grid layout
* 🔹 Filter products by category
* 🔹 Search products by name (real-time)
* 🔹 Sort products by price (Low → High / High → Low)
* 🔹 View detailed product information
* 🔹 Add to cart (UI-based counter)
* 🔹 Error handling for API failures
* 🔹 Responsive design (mobile, tablet, desktop)

## 🧠 Tech Stack

* HTML
* CSS (Bootstrap)
* JavaScript (Vanilla JS)

## ⚠️ Challenges Faced

* Managing filtered vs original product data
* Implementing real-time search without breaking filters
* Handling asynchronous API calls properly
